## Install steps

```
mkdir <directory>
cd <directory>
npm create vite@latest .
```

Options:
* Framework: React
*  Variant: JavaScript

```
npm install
npm run dev
```

Other things:
* Remove most stuff from files in `src` folder
* Make `src/components` directory
* Optional: `src/styles` directory

* Create components
  * `src/components/Box`
  * `src/components/Box/Box.jsx`
  * `src/components/Box/box.css`
